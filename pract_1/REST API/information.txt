У даному розділі курсу розбирається приклад використання API для побудови простого ETL pipline для роботи з даними.

Для роботи з API пропонується використати один з варіантів відкритих API
(https://mixedanalytics.com/blog/list-actually-free-open-no-auth-needed-apis/),
що віддають json файл у відповідь на запит.
